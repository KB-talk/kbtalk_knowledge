相位法测角

两天线相位法测角基本原理

利用电磁波在空间中传播到不同位置接收天线时产生的相位差，来反推目标的方向（角度）。

其本质是将空间上的距离差，转化为相位差，再通过测量相位差来解算角度。

电磁波每传播一个波长 λ\lambdaλ 的距离，相位变化2π2\pi2π。

φ=2πdsinθλ\varphi =\frac{2\pi dsin\theta }{\lambda } φ=λ2πdsinθ​

θ=arcsin(φλ2πd)\theta =arcsin(\frac{\varphi \lambda }{2\pi d} ) θ=arcsin(2πdφλ​)

提示

短基线保证大的无模糊测角范围。

长基线保证高的测角精度。

等信号法

提示

两天线方向图相同，交叠位置大约在3分贝处。

工作原理

  • 比幅法

天线波束：天线产生两个完全相同的波束，它们在空间上关于天线轴对称，并有一部分重叠。重叠区域的中心线就是“等信号轴”。

波束A的方向图函数为 F(θ−θ0)F(\theta - \theta_0)F(θ−θ0​)

波束B的方向图函数为 F(θ+θ0)F(\theta + \theta_0)F(θ+θ0​)

其中 θ0\theta_0θ0​ 是单个波束最大值方向与等信号轴（θ=0\theta=0θ=0）的夹角。

信号接收：当目标位于方向 θ\thetaθ 时（θ\thetaθ 是相对于等信号轴的偏角），两个通道接收到的信号电压幅度分别为：

uA=kF(θ−θ0)u_A = k F(\theta - \theta_0)uA​=kF(θ−θ0​)

uB=kF(θ+θ0)u_B = k F(\theta + \theta_0)uB​=kF(θ+θ0​)

kkk 是一个与距离、目标反射率等有关的常数。

计算幅度比：系统计算两个信号的幅度比（或差值）。比值 RRR 与目标的偏角 θ\thetaθ 有确定的函数关系：

R=uAuB=F(θ−θ0)F(θ+θ0)R = \frac{u_A}{u_B} = \frac{F(\theta - \theta_0)}{F(\theta + \theta_0)}R=uB​uA​​=F(θ+θ0​)F(θ−θ0​)​

或者使用差值：Δ=uA−uB\Delta = u_A - u_BΔ=uA​−uB​

测角：

判别方向：如果 uA>uBu_A > u_BuA​>uB​ (R>1R>1R>1, Δ>0\Delta>0Δ>0)，说明目标偏在等信号轴的A侧；反之，则偏在B侧。

判别大小：比值 RRR 的大小直接反映了目标偏离等信号轴的角度 θ\thetaθ 的大小。通过预先标定好的 R−θR-\thetaR−θ 曲线（称为“角误差特性曲线”），就可以精确计算出偏角 θ\thetaθ 的值。

实现等信号：在自动跟踪系统中，这个误差信号 Δ\DeltaΔ 会被反馈给伺服系统，驱动天线转动，使 Δ\DeltaΔ 趋向于零。当天线对准目标时，θ=0\theta=0θ=0，则 uA=uBu_A = u_BuA​=uB​，Δ=0\Delta=0Δ=0，天线停止转动。

优点：原理简单直观。 缺点：测角精度依赖于两个通道（放大器、接收机等）的幅度一致性。如果两个通道的增益特性有微小差异，就会直接带来测角误差。

  • 和差法

和差法同样使用两个重叠的波束，但它利用微波网络（如魔T、双T、3dB电桥等）形成“和”信号（Σ\SigmaΣ）与“差”信号（Δ\DeltaΔ），从而将角误差信息蕴含在信号的相位中，而相位比对幅度的一致性不敏感。

天线波束与和差网络：

两个馈源（或阵列）分别产生波束A和B。

通过一个微波和差比较器（ magic-T ）：

和通道：Σ=uA+uB\Sigma = u_A + u_BΣ=uA​+uB​

差通道：Δ=uA−uB\Delta = u_A - u_BΔ=uA​−uB​

信号特性：

和信号 Σ\SigmaΣ：其方向图是两个波束的叠加。在等信号轴方向 (θ=0\theta=0θ=0) 有最大值，用于测距和作为相位参考基准。

差信号 Δ\DeltaΔ：其方向图呈奇对称。在等信号轴方向 (θ=0\theta=0θ=0) 为零值（谷点），越偏离零点，信号幅度越大。其相位（0°或180°）则代表了偏离的方向。

测角原理：

当目标偏离等信号轴时，Δ≠0\Delta \neq 0Δ=0。

判别方向：将差信号 Δ\DeltaΔ 与和信号 Σ\SigmaΣ 进行相位检波。

如果目标偏在A侧，则 Δ\DeltaΔ 与 Σ\SigmaΣ 同相。

如果目标偏在B侧，则 Δ\DeltaΔ 与 Σ\SigmaΣ 反相（相位差180°）。

判别大小：误差电压的大小正比于 ∣Δ∣/∣Σ∣|\Delta| / |\Sigma|∣Δ∣/∣Σ∣，即差信号与和信号的幅度比。这个比值与偏角 θ\thetaθ 成正比。

误差电压 Verror∝∣Δ∣∣Σ∣≈kθV_{error} \propto \frac{|\Delta|}{|\Sigma|} \approx k \thetaVerror​∝∣Σ∣∣Δ∣​≈kθ （在小角度下近似线性）

实现等信号：自动跟踪系统根据误差电压的极性（相位） 判断驱动方向，根据大小判断驱动速度，最终使天线对准目标，此时 Δ=0\Delta = 0Δ=0，误差电压为零。

优点：

高精度：利用相位信息判别方向，对接收机通道的幅度增益一致性要求大大降低。

高数据率：理论上可以从一个脉冲回波中同时获得“和”、“差”信息（故名“单脉冲”），避免了目标起伏带来的误差，数据率高，适合跟踪高速目标。

缺点：系统复杂，需要和差网络和多通道接收机（至少需要和、差两个通道）。

特点

优点：便于角度跟踪

缺点：最大作用距离小于最大信号法
